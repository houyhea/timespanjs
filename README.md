#timespanjs
===
##简介
实现日期差值的人性化显示的javascript库。比如，可以显示成：2年2个月2小时4分钟23秒。同时支持国际化,可扩展。
采用[MIT 许可协议](https://github.com/houyhea/timespanjs/blob/master/LICENSE)。
